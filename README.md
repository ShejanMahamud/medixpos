## What is MedixPOS?

MedixPOS is a complete pharmacy management solution designed for modern pharmacies. From inventory tracking to sales processing, customer management to financial reporting - everything you need to run your pharmacy efficiently in one desktop application.

## Why MedixPOS?

**Purpose-Built for Pharmacies**
Designed specifically for pharmacy workflows with prescription tracking, batch control, and expiry management. Not a generic POS adapted for pharmacy use.

**Complete Offline Operation**
All data stored locally on your computer. No monthly subscriptions, no cloud dependency, no internet required for daily operations. Your data stays with you.

**Fast and Reliable**
Desktop performance with instant product search, real-time inventory updates, and responsive interface. No lag, no loading screens.

**Professional Grade**
Secure license management, encrypted data storage, comprehensive audit trails, and role-based access control. Enterprise features at pharmacy scale.

**Easy to Use**
Clean, intuitive interface that your staff will master in minutes. No lengthy training required.

**Fast Checkout Experience**

## Installation

### System Requirements

**Windows**

- Windows 10 or 11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 500MB free disk space
- Internet connection (for license activation only)

### Quick Install

**Windows**

1. Download the latest [MedixPOS installer](https://github.com/ShejanMahamud/pharmacy-pos/releases/latest)
2. Run the `medixpos-1.0.0-setup.exe` file
3. Follow the installation wizard
4. Launch MedixPOS from Start Menu

**Note**: Windows SmartScreen may show a warning for unsigned applications. Click "More info" and then "Run anyway" to proceed with installation.

## Getting Started

### License Activation

1. Launch MedixPOS
2. Enter your license key when prompted
3. The application will verify and activate your license
4. You're ready to start using the system

### First Login

After successful license activation, the system will display default administrator credentials on first launch. Use these to log in and change the password immediately.
